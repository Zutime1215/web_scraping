1. Python3 must be installed in your machine.
2. Open your Terminal/Command Promt and type this command "pip install -r requirements.txt" (without " ").
3. If you want to create your own room, Run the "menu.py" file and follow the steps.
4. If you have any channel's url and key, then edit the "configme.txt" file and replace <<<>>> with your name, url and key.

Enjoy!
